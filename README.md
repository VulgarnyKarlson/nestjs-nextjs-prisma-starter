# NestJS NextJS Prisma Starter

## Installation

```bash
$ docker-compose up -d # start prisma and postgres
$ npm install
$ npm run prisma:dev
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
